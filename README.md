# Clicky Crates – Prototype 5
This project is Prototype 5 from my Unity learning journey.
It is based on Unity’s Create with Code course.

## Features

- Start screen with multiple difficulty options (Easy, Medium, Hard)
- Volume slider to control overall sound levels
- Pause menu (ESC) with restart functionality
- Click‑to‑destroy gameplay: crates break when hovering over them
- Trail renderer for visual feedback
- Simple score and lives display
- Sound effects recorded with my own voice (pitched and processed)
- Background music from the Unity Asset Store (Free – Casual & Relaxing Game Music Pack)


## Purpose
This prototype focuses on:
- learning basic UI systems
- implementing difficulty selection and audio controls
- creating a pause and restart flow
- experimenting with simple hover‑to‑destroy interactions
